# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it privately.

- Preferred: open a GitHub Security Advisory (Security → Advisories → Report a vulnerability)
- Alternative: email `jiahao15345932820@gmail.com`

Please include:

- A clear description of the issue and impact
- Steps to reproduce (PoC)
- Affected versions/commit hash
- Any proposed mitigation

We will acknowledge receipt and work on a fix as quickly as possible.
