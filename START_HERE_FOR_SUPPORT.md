## What YNX is

YNX is an early open-source infrastructure project building AI-native
execution and developer-facing Web4 tooling.

Current boundary:

- YNX has real public technical artifacts and live public-testnet evidence.
- YNX is not yet a publicly announced legal operating entity.
- YNX should not be described as a mainnet-grade production network, custody
  platform, or completed institutional compliance stack.
