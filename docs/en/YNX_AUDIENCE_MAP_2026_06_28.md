## 1. Why this file exists

YNX now has enough material that the problem is no longer "nothing exists."

The problem is that different readers should not start from the same file:

- developers need runnable surfaces and APIs
- grant reviewers need truthful public-testnet evidence and concise framing
- providers need YNX Card readiness and compliance boundaries
- operators/founders need latest truth, blockers, and closeout packs
- community readers need the shortest safe public story

This file is the routing layer for those audiences.

If the reader has not yet been pre-qualified by audience, start with:
