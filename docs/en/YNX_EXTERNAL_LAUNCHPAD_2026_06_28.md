## 2. The shortest honest description of YNX

YNX is an early open-source Web4 and AI-execution infrastructure project with
live public-testnet surfaces, policy-bounded execution flows, AI settlement
rails, accountability tooling, and a programmable card-control direction.

Safe current boundary:

- live public-testnet infrastructure exists
- developer, AI, Web4, trace, and mock-card logic all have real repository
  artifacts and runnable surfaces
- YNX Card is currently a programmable mock/control foundation, not a live
  issuer-backed card product
- YNX should not yet be described as a formed operating company, a completed
  legal/compliance stack, or a production custody/payment network
