# YNX v2 本地完整开发运行手册

状态：active  
最后更新：2026-03-07

本手册只用于本地完整开发，不涉及服务器迁移。

## 1. 一条命令跑完整闭环

```bash
cd ~/Desktop/YNX/chain
./scripts/v2_local_complete.sh all
```

该命令会执行：

1. v2 初始化
2. 启动全栈服务
3. verify + smoke 验收
4. 打包 release 产物

## 2. 日常迭代命令

```bash
cd ~/Desktop/YNX/chain
./scripts/v2_local_complete.sh up
./scripts/v2_local_complete.sh verify-smoke
./scripts/v2_local_complete.sh pack
./scripts/v2_local_complete.sh down
```

## 3. 本机多验证人仿真

```bash
cd ~/Desktop/YNX/chain
YNX_VALIDATOR_COUNT=6 ./scripts/v2_local_complete.sh multinode
```

## 4. 本地完成标准

- `v2_public_testnet_verify.sh` 输出 `PASS`
- `v2_public_testnet_smoke.sh` 输出 `PASS`
- `v2_testnet_release.sh` 生成打包与校验文件
- 多验证人仿真可持续出块
