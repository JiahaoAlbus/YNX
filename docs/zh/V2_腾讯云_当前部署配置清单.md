## 入站端口（腾讯云）

腾讯云 Lighthouse 防火墙（IPv4）至少需要放行：

- TCP `80`，来源 `0.0.0.0/0`（ACME / 跳转）
- TCP `443`，来源 `0.0.0.0/0`（HTTPS 网关）
- TCP `36656`，来源 `0.0.0.0/0`（YNX P2P）

主机本机监听端口（期望值）：

- 节点：`36656`（P2P）、`36657`（RPC）、`31317`（REST）、`39090`（gRPC）、`38545`（EVM RPC）、`38546`（EVM WS）
- 应用服务：`38080`（faucet）、`38081`（indexer）、`38082`（explorer）、`38090`（AI gateway）、`38091`（Web4 hub）
