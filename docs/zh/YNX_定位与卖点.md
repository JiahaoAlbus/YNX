# YNX 定位与卖点（中文）

状态：active  
最后更新：2026-02-17

## 一句话定位

YNX 是一条**治理优先（governance-native）的 EVM 链**，面向真实 Web3 服务场景。

## 用户为什么选择 YNX

- 主网同逻辑公测：公测流程与未来主网逻辑对齐，迁移成本低。
- 治理透明：治理与费率分配信息可通过 API 读取，不是黑箱运营。
- 上手快：节点部署和运维有可复制命令与一键验收脚本。
- 生态早期红利：开发者和验证人参与门槛低、协作效率高。

## 与大链的差异

YNX 不以“当前最大流动性”作为卖点，核心差异在：

- 规则执行透明，
- 运维接入清晰，
- 治理信息可验证，
- 早期参与方可获得更高话语权。

## 机器可读入口

- `GET /ynx/overview`

可返回：

- 治理地址与费率参数，
- 链定位与价值主张字段，
- 当前链运行概览。
