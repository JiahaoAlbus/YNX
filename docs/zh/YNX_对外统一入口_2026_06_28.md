## 2. 当前最短且诚实的 YNX 描述

YNX 是一个早期的开源 Web4 / AI execution 基础设施项目，已经有 live
public-testnet surface、policy-bounded execution flow、AI settlement rail、
accountability tooling，以及面向未来 provider 的 programmable card-control
方向。

当前安全边界：

- live public-testnet infrastructure 已存在
- developer、AI、Web4、trace、mock-card 逻辑都已经有真实仓库产物和可运行 surface
- YNX Card 目前是 programmable mock/control foundation，不是 live issuer-backed card
- 现在不能把 YNX 描述成已完成公司化、已完成法律/合规、或生产级 custody/payment network
