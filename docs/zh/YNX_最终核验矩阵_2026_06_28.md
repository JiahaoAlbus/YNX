# YNX 最终核验矩阵

状态：active verification matrix  
整理日期：2026-06-28  
用途：定义在现阶段把 YNX 说成 this phase 已完成之前，必须核对哪些东西

这不是营销摘要。

这是最后收口时的实际核验网格，用来检查：

- live public-testnet 真相
- 本地 / repo 可运行真相
- mock / testnet / provider / legal 边界
- 还没完成的外部阻塞项

## 1. 核验规则

在说 YNX 现阶段“done”之前，每个 surface 都要同时核对：

1. 可运行代码或 live endpoint
2. 对应公开文档
3. 对应 pack / manifest 引用
4. truthful boundary wording
5. 如果还没 fully live，必须有明确 blocker 列表

只要这几项有任何一项不对齐，这个 surface 就不能算 fully verified。
