# YNX Web4 公开测试网三分钟演示提纲

## 目标

做一个三分钟 sponsor demo，证明系统不是概念，而是已经上线、受保护、可测量。

## 演示结构

### 1. 开场：YNX 是什么

- 打开 `https://www.ynxweb4.com`
- 用一句话介绍：
  - `YNX is a live public testnet for protected AI onchain execution and cross-chain test-asset routing.`

### 2. 网络真实在线证明

- 打开 `https://www.ynxweb4.com/api/network/status`
- 说明 RPC、EVM、REST、faucet、indexer、explorer、AI、Web4 都在线
- 点出 validator set 已恢复为 4 个 bonded validators

### 3. AI 保护能力证明

- 打开 `/ai`
- 解释 Web4 session policy 和受保护的 `trade.execute`
- 展示 AI preflight / settlement，但不要把 autonomy 说得过头

### 4. Bridge readiness 证明

- 打开 `/readiness`
- 明确讲：
  - `4/5 deposit-tested`
  - `5/5 some form of release proof observed`
  - `2/5 automatic-loop-ready`
  - `2/5 strongest observed automatic public evidence today`
- 把 BSC 讲成最明确的自动化阻塞项，把 BTC/TRON 讲成“配置已具备、但公开自动化实证还没有 Sepolia 硬”

### 5. 结尾：要什么支持

- 明确申请的是 grant / sponsorship，不是 token 融资
- 资金用途聚焦：
  - infra 与 adapter completion
  - audit 与 reliability
  - builder onboarding 与 ecosystem demo

## 建议收尾话术

`We are not pitching a concept. The public testnet is already live, protected, and measurable. We are asking for ecosystem funding to complete the remaining route automation and production hardening.`
