# YNX 开放测试网下一步执行清单

状态：Active  
最后更新：2026-02-21  
规范语言：中文

## 1）冻结当前稳定基线

```bash
cd ~/YNX/chain
./scripts/testnet_freeze_tag.sh
```

该命令会在 `ops-logs/freeze/` 生成冻结报告，并打一个带时间戳的 git tag。

## 2）开放验证人接入

对外发布这两份文档：

- `README.md`（Path B/C/D/G/H/I）
- `docs/en/VALIDATOR_ONBOARDING_PACKAGE.md`

每个新验证人统一走安全加入流程：

```bash
cd ~/YNX/chain
./scripts/validator_onboard_safe.sh
```

## 3）持续监控与告警

至少在一台运维机常驻 watchdog：

```bash
cd ~/YNX/chain
./scripts/testnet_watchdog.sh
```

需要推送告警时：

```bash
ALERT_WEBHOOK_URL="https://your-webhook-endpoint" ./scripts/testnet_watchdog.sh
```

## 4）保持公网入口可用

至少保持：

- 1 个公网 RPC
- 1 个公网浏览器
- 1 个水龙头

快速自检：

```bash
cd ~/YNX
./chain/scripts/public_testnet_verify.sh
```

## 5）主网上线前门槛

满足以下项再推进主网：

- 验证人集合分布化（非单运营方控制）
- 连续稳定出块窗口内无严重事故
- 监控与应急流程演练通过
- 开放测试网阶段文档与接入流程已验证可用
