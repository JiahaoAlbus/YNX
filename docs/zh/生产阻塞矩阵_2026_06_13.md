# YNX 生产阻塞矩阵

状态日期：2026-06-13

## 当前生产状态

- 验证人 bonded：`4/4`
- 公网 readiness gate：`PASS`
- bridge full-loop-tested：`5/5`
- automatic-loop-ready：`2/5`
- automatic-loop-observed：`2/5`

重要解释：

- `4/4 bonded` 本身不等于强验证人独立性；
- `5/5 full-loop-tested` 不等于 `5/5 production-safe`；
- `automatic_loop_ready` 表示 route 配置和 adapter 已具备；
- `automatic_loop_observed` 表示已经有真实 watcher / release 证据，不只是配置到位；
- 这份矩阵应被理解成 testnet 运行证据，而不是“已经机构级”的证明。
